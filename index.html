Element

Types of Elements:

Inline level Element

Displayed in Sameline and we cannot adjust Height and Width. 
Eg: <b>, <i>, <mark>, etc.,

Block Level Element

It will take the entire width of the page and can able to adjust the Height and Width.
Eg: <h1>, <h2>, <p> etc.,

Inline-Block Level Element

Combination of both display in same line and we can adjust height and width 
Eg: <img>, <input>, <button>, etc.,

Attributes: They proide aditional information or functionallity to HTML Element.

Syntax: 
          <element attribute = "value">content</element>
