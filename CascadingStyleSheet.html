Simple Selector

This is the most basic selectors

This selectors will target element based on their tag name, class, id, universal, grouping

Universal Selector

This selector targets all the elements

Example: 

   
               *{  
                    margin: 10px;
               }

Element/Tag Name Selector

This selector target an element based on tag name

Example: 

               p{
                    color: orangered;
               }

Grouping Selector

This Selector group multiple selector

Example: 

               h1,pre{
                    Color: blueviolet;
               }

id Selector

This selector target an element with id and id cannot be same

Example: 

               #one{
                    Color: darkolivegreen;
               }

Class Selector

This Selector target element with the specific slass name. Class name can be repeated same element can have multiple classes seperated by spaces

Example: 

               .sss{
                    Color: darkred;
               }

CSS Simple Selector Specificity Priority

inline>id>class>tagname, grouping>universal>

Decendent Selector

These Selector target all children, grandchildren, great grandchildren, great great grandchildren, etc.,

Descendent Selector - (Space)

Syntax: ancestor decendant

Example: 

               div h2{
                    Color: darkblue;
               }

target all the h1 present inside div.

Direct Child Selector

These Selector targets only the direct child of the element not grandchild great grandchild, etc.,

Direct Child Selector - (>)

Syntax: Parent>Child

Example: 

               div>p{
                    Color: darkslategray;
               }

target only the p presint at the first level of div.

Adjacent Sibling Selection

This selection target the sibling immediatly right to the specified element.

It only target sibling immidiate right not the descendant

Adjacent Sibling Selector - (+)

Example 

               pre+p{
                    Color: darkgreen;
               }

it targets p immediate right to pre

General Sibling Selector

This selector target all the sibling following right to the specified element

General Sibling Selector - (~)

Example: 

               h2~h3{
                    color: darkred;
               }
               
Target all h3 which are right sibling of h2

Pseudoclass Selector

These selector are used to target element based upon a specific condition.

These selector have prefix with the single colon(:)

User interaction states

:hover - When the mouse is over the element.

Example: button:hover

:active - When the element is being clicked

Example: a:active

:focus - When an input is focused (clicked/tab).

Example: input:focus

:visited - A link that was already visited.

Example: a:visited

:link - A link that hasn't been clicked yet.

Example: a:link

Structural/Position Selectors

:first-child - Selects the first child of a parent.

Example: li:first-child

:last-child - Selects the last child of a parent.

Example: li:last-child

:nth-child(n) - Selects the nth child.

Example: li:nth-child(3)

:nth-last-child(n) - Counts form the end.

Example: li:nth-last-child(2)

:only-child - Select it it's only child.

Example: p:only-child

Form State Selectors

:checked - Checked ratio or checkbox

Example: input:checked

:disabled - Disabled form input

Example: input:disabled

:enabled - Enabled form input.

Example: input:enabled

:required - input with required attribute.

Example: input:required

:optional - input without required.

Example: input:optional

:valid - Valid input(like correct email format)

Example: input:valid

:invalid - Invalid input(like empty required field)

Example: input:invalid

:read-only - Read-only input.

Example: input:read-only

:read-write - Editable input.

Example: input:read-write

Negation and other

:empty - Select elements with no children/content

Example: div:empty

:root - Seects the html element

Example: :root{--main: red;}

Pseudo-Element Selector

These selectors are used to style specific parts of an element's content.

Pseudo-elements have a prefix with a double colon (::)

Common Pseudo-Elements

::before - Adds content before the element.

Example: p::before { content: "→ "; }

::after - Adds content after the element.

Example: p::after { content: " ✔"; }

::first-letter - Styles the first letter of an element's text.

Example: p::first-letter { font-size: 2em; }

::first-line - Styles the first line of an element's text.

Example: p::first-line { color: blue; }

::selection - Styles the part of text selected by the user.

Example: ::selection { background: yellow; }

::placeholder - Styles the placeholder text of input fields.

Example: input::placeholder { color: gray; }

::marker - Styles the bullet or number in a list item.

Example: li::marker { color: red; }

::file-selector-button - Styles the file upload button of an input field.

Example: input::file-selector-button { background-color: lightblue; }

Attribute Selectors

These selectors are used to target elements based on their attributes or attribute values.

They use square brackets [] around the attribute name.

Basic Attribute Selector

[attr] – Selects elements that have the given attribute.

Example: input[required]

Exact Match

[attr="value"] – Selects elements with the exact attribute value.

Example: input[type="text"]

Contains Word

[attr~="value"] – Selects elements where the attribute contains a word (space-separated).

Example: div[class~="box"]

Starts With

[attr^="value"] – Selects elements where the attribute value starts with a string.

Example: a[href^="https"]

Ends With

[attr$="value"] – Selects elements where the attribute value ends with a string.

Example: img[src$=".jpg"]

Contains Substring

[attr*="value"] – Selects elements where the attribute value contains a string.

Example: a[href*="login"]
