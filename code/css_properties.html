text properties

     Text properties are used to apply various effects and styles to text content. Affect how text is positioned and arranged on a page


1. color

     Changes the color of the text.

Syntax: color: value;

Values: Any valid color (e.g., red, #ff0000, rgb(255, 0, 0)).


2. text-align

     Aligns the content horizontally based on the specified value.

Syntax: text-align: value;

Values: left, right, center, justify, start, end.


3. text-transform

     Converts the text to a specific format (e.g., uppercase).

Syntax: text-transform: value;

Values: lowercase, uppercase, capitalize.


4. text-decoration

     Adds decorations such as underline, overline, or strikethrough.

Syntax: text-decoration: line style color;

Values:

  • line: underline, overline, line-through, none
  • style: solid, dashed, dotted, double, wavy
  • color: any valid color


5. text-indent

     Adds indentation at the beginning of the text content.

Syntax: text-indent: value;

Values: Any valid length (e.g., 20px, 2em).


6. text-wrap

     Controls whether the text should wrap to the next line or also use white-space.

Syntax: text-wrap: value;

Values: wrap, nowrap.


7. letter-spacing

     Sets the space between individual letters in the text.

Syntax: letter-spacing: value;

Values: Any valid length (e.g., 2px, 0.1em).


8. word-spacing

     Sets the space between words in the text.

Syntax: word-spacing: value;

Values: Any valid length (e.g., 4px, 0.5em).


9. line-height

     Specifies the vertical spacing between lines of text.

Syntax: line-height: value;

Values: Normal number or length (e.g., 1.5, 20px).


10. text-shadow

     Applies shadow effect to the text.

Syntax: text-shadow: x-offset y-offset blur-radius color;

Example: text-shadow: 2px 2px 3px blue;


11. font-family

     Defines the font typeface for the text.

Syntax: font-family: value;

Values: One or more font names, followed by a generic family.
Examples: Arial, "Times New Roman", sans-serif, serif, monospace.


12. font-size

     Specifies the size of the font.

Syntax: font-size: value;

Values: Can be in absolute (px, pt) or relative units (em, rem, %, vw, etc.)
Examples: 16px, 1.5em, 120%, 2rem.


13. font-weight

     Controls the boldness (thickness) of the text.

Syntax: font-weight: value;

Values:

  • Keywords: normal, bold, lighter, bolder
  • Numeric values: 100 (thin) to 900 (extra bold), where 400 = normal and 700 = bold


CSS Units

     In CSS, units specify the length of an element or the size of its content. Units are categorized into two types:

     In CSS, the default size of any text is 16px


1. Absolute Units

     These units are static and do not depend on any other values (fixed sizes).

Common Absolute Units and Conversion:

  • px → Pixels (1px = 1/96 inch)
  • pt → Points (1pt = 1/72 inch ≈ 1.33px)
  • pc → Picas (1pc = 12pt = 16px)
  • in → Inches (1in = 2.54cm = 96px)
  • cm → Centimeters (1cm ≈ 37.8px)
  • mm → Millimeters (1mm ≈ 3.78px)

2. Relative Units

     These units are dynamic and depend on other values like parent or root elements.

Common Relative Units:

  • % → Depends on the parent element. (e.g., width: 50%)
  • em → Depends on the parent element’s font-size (e.g., 2em = 2 × parent font-size)
  • rem → Depends on the root element’s font-size (default = 16px | If you set the root font size to 20px: 1rem = 20px, 2rem = 2 * 20px = 40px, 0.5rem = 0.5 * 20px = 10px)
  • vw → 1% of the viewport width
  • vh → 1% of the viewport height
  • vmin → 1% of the smaller between viewport height and width
  • vmax → 1% of the larger between viewport height and width

Which Unit to Use?

  • Use px for precise control.
  • Use rem/em for responsive typography.
  • Use vw/vh for full-screen sections.
  • Use % for layouts that depend on parent size.
  • Use pt/cm for print-specific designs.

Background Properties

     Background properties are used to apply effects to an element's background. The background shorthand property combines background-related properties into a single declaration (except background-size).


1. background

     A shorthand property for background properties (excluding background-size).

Syntax: background: color url() repeat position attachment;

Instead of writing all properties individually, you can specify all values in a single line.


2. background-color

     Specifies the background color of an element.

Syntax: background-color: value;

Values: Any valid color (e.g., red, #ff0000, rgb(255, 0, 0)).


3. background-image

     Specifies an image to use as the background of an element.

Syntax: background-image: url("path/to/image");

If the image is small, it will repeat; if large, it may get cropped to fit the container.


4. background-repeat

     Defines how a background image is repeated.

Syntax: background-repeat: value;

Values:

  • repeat → Repeats the image horizontally and vertically (default)
  • no-repeat → Shows the image only once
  • repeat-x → Repeats the image horizontally
  • repeat-y → Repeats the image vertically


5. background-position

     Specifies the starting position of a background image.

Syntax: background-position: x-position y-position;

Values:

  • x-position → left, center, right
  • y-position → top, center, bottom


6. background-size

     Defines the scaling of the background image within the container.

Syntax: background-size: value;

Values:

  • auto → Default. Keeps the original image size
  • contain → Scales the image to fit the container (no cropping)
  • cover → Scales the image to fill the container (may crop)


7. background-attachment

     Specifies whether the background image scrolls with the page or stays fixed.

Syntax: background-attachment: value;

Values:

  • scroll → Background moves when the page is scrolled (default)
  • fixed → Background remains fixed in place even when scrolling


Color Formats

     In CSS, we can apply colors in many ways. All colors are created from a combination of Red, Green, and Blue (RGB). Below are the different ways to apply colors in CSS:


1. Color Name

     Applies a color using its predefined name.

Syntax: property: color-name;

Values: Any valid color name (e.g., red, blue, green).


2. RGB (Red, Green, Blue)

     Uses numeric values from 0 to 255 for each color component.

Syntax: property: rgb(red, green, blue);

Examples:

  • background-color: rgb(0, 0, 0); → black
  • background-color: rgb(255, 255, 255); → white


3. RGBA (Red, Green, Blue, Alpha)

     Similar to RGB but includes an alpha channel for opacity.

Syntax: property: rgba(red, green, blue, alpha);

Alpha value ranges from 0.0 (fully transparent) to 1.0 (fully opaque).
Example: background-color: rgba(255, 0, 0, 0.5);


4. HEX (Hexadecimal)

     Represents color using hexadecimal values for red, green, and blue.

Syntax:

  • Full: property: #rrggbb;
  • Short: property: #rgb; (each digit is doubled)
Examples:
  • background-color: #aabdc1;
  • background-color: #fff; → white
  • background-color: #000; → black


5. HSL (Hue, Saturation, Lightness)

     Defines colors using the HSL model (Hue, Saturation, Lightness).

Syntax: property: hsl(hue, saturation, lightness);

  • Hue: Angle on color wheel (0°–360°), e.g., 0° = Red, 120° = Green, 240° = Blue
  • Saturation: Intensity of color (0%–100%)
  • Lightness: Brightness (0% = black, 50% = normal, 100% = white)


6. HSLA (Hue, Saturation, Lightness, Alpha)

     Similar to HSL but includes alpha for opacity control.

Syntax: property: hsla(hue, saturation, lightness, alpha);

Alpha value ranges from 0.0 (transparent) to 1.0 (opaque).


Overflow Property

     The overflow property in CSS controls how content is displayed when it exceeds the size of its container. It is commonly used to manage scrollbars and prevent content overflow.

Syntax:

selector {
    overflow: value;
}

Values of Overflow Property

1. visible (Default)

     Content overflows the container and is visible outside.

Example:
div {
    width: 200px;
    height: 100px;
    overflow: visible;
}

Content may exceed the container, leading to design issues.


2. hidden

     Hides any content that overflows the container. No scrollbars appear.

Example:
div {
    width: 200px;
    height: 100px;
    overflow: hidden;
}

Useful for preventing content from spilling out.


3. scroll

     Adds scrollbars (horizontal and vertical) even if the content fits.

Example:
div {
    width: 200px;
    height: 100px;
    overflow: scroll;
}

Ensures users can scroll to see hidden content.


4. auto

     Adds scrollbars only if needed (when content overflows).

Example:
div {
    width: 200px;
    height: 100px;
    overflow: auto;
}

Better for usability as it avoids unnecessary scrollbars.


Shorthand Usage (overflow-x and overflow-y)

     The overflow property can be applied separately for horizontal and vertical scrolling.

div {
    overflow-x: scroll;   /* Horizontal scrollbar */
    overflow-y: hidden;   /* No vertical overflow */
}

Gradients

     CSS gradients let you display smooth transitions between two or more specified colors.

There are three types of gradients:

  • Linear Gradients (goes down/up/left/right/diagonally)
  • Radial Gradients (defined by their center)
  • Conic Gradients (rotated around a center point)

Linear Gradient

     A linear gradient transitions colors along a straight line, either horizontally, vertically, or at an angle.

Syntax:
background-image: linear-gradient(direction, color-1, color-2, ...);

direction: to bottom (default), to top, to right, to left, to bottom right, to top left, to top right, to bottom left.

Radial Gradient

     A radial gradient transitions colors radiating outward from a center point.

We can also set the shape of the radial gradient as ellipse or circle

div {
    background-image: radial-gradient(circle, red, yellow, green);
}

Conic Gradient

     A conic gradient is a gradient with color transitions rotated around a center point.

div {
    background-image: conic-gradient(red, yellow, green);
}


Border

     Border properties allow you to customize the borders of elements with various styles, colors, and widths.

1. border

     It is a shorthand property for border-width, border-style, and border-color. The border-style is mandatory.

Syntax:
border: border-width border-style border-color;

2. border-width

Syntax:
border-width: value;

3. border-style

Syntax:
border-style: value;
/* Values: solid, dashed, dotted, double, groove, ridge, none */

4. border-color

Syntax:
border-color: value;

5. border-top / border-bottom / border-left / border-right

Used to style borders for specific sides of an element

Syntax:
border-top: border-width border-style border-color;
border-bottom: border-width border-style border-color;
border-left: border-width border-style border-color;
border-right: border-width border-style border-color;

6. border-collapse

Used for tables to decide whether the table cells' borders should be separated or collapsed.

Syntax:
border-collapse: value;
/* Values: separate, collapse */

7. border-spacing

Specifies the space between cells in a table.
Only works when border-collapse is set to separate.

Syntax:
border-spacing: value;


Border Radius

     The border-radius property allows you to create rounded corners for elements by defining the radius of their corners.

Individual Corner Radius

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius

Shorthand Property:

border-radius: <top-left> <top-right> <bottom-right> <bottom-left>;

Examples:

  • 1 value: Applies to all four corners → border-radius: 10px;
  • 2 values: top-left & bottom-right, top-right & bottom-left → border-radius: 10px 20px;
  • 3 values: top-left, top-right & bottom-left, bottom-right → border-radius: 10px 20px 30px;
  • 4 values: top-left, top-right, bottom-right, bottom-left → border-radius: 10px 20px 30px 40px;


Box Model

  • The CSS box model describes how elements are displayed and how spacing is calculated.
  • It consists of 4 parts:
    • Content Box
    • Padding Box
    • Border Box
    • Margin Box

Content Box

  • This is the area where content (text, images) is displayed.
  • Only content height and width are included here.

Padding Box

  • This is the space between content and border.
  • It increases space inside the box.
  • Example:
    • padding: 10px;

Border Box

  • It defines the border around an element.
  • Example:
    • border: 2px solid blue;

Margin Box

  • It is the space outside the border.
  • It separates the element from others.
  • Example:
    • margin: 10px;

Margin Shorthand Property

  • Used to set all four sides' margins in one line.
  • Four values: margin: 10px 20px 30px 40px;
    • Top: 10px
    • Right: 20px
    • Bottom: 30px
    • Left: 40px
  • Three values: margin: 10px 20px 30px;
    • Top: 10px
    • Left & Right: 20px
    • Bottom: 30px
  • Two values: margin: 10px 20px;
    • Top & Bottom: 10px
    • Left & Right: 20px
  • One value: margin: 10px;
    • All sides: 10px
  • Same format is used for padding also.

Height and Width Calculation

  • Total height = margin-top + border-top + padding-top + content-height + padding-bottom + border-bottom + margin-bottom
  • Total width = margin-left + border-left + padding-left + content-width + padding-right + border-right + margin-right

Important Note

  • If we use:
    • height: 100px;
    • padding: 10px;
  • Total height = 100 + 10 + 10 = 120px
  • To fix the box size, use:
    • box-sizing: border-box;

Resetting Default Styles

  • To remove default margin and padding:
  • * {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

box-sizing Property

  • The box-sizing property controls how the total width and height of an element are calculated.
  • It decides whether padding and border are included in the element's total size.

Values of box-sizing

1. content-box (Default)

  • This is the default value.
  • The width and height only apply to the content.
  • Padding and border are added outside the set width/height.
  • Example: 
    width: 200px;
padding: 20px;
border: 5px solid;
box-sizing: content-box;
  • Total width = 200 + 20 + 20 + 5 + 5 = 250px

2. border-box

  • The width and height include content, padding, and border.
  • Padding and border are subtracted from the specified width/height.
  • More useful when you want fixed box sizes.
  • Example: 
    width: 200px;
padding: 20px;
border: 5px solid;
box-sizing: border-box;
  • Total width remains 200px (internally adjusted)

Usage (Reset all elements)

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
  • This makes layout predictable by including padding and border in element size.
