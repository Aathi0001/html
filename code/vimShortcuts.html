Vim Shortcuts Cheat Sheet

Normal Mode (Navigation / Editing)

ShortcutAction
hMove left
jMove down
kMove up
lMove right
0Go to beginning of line
$Go to end of line
wJump to next word
bJump to beginning of word
ggGo to first line
GGo to last line
Ctrl+fPage down
Ctrl+bPage up
ddDelete current line
dwDelete word
xDelete character under cursor
uUndo last action
Ctrl+rRedo
yyYank (copy) line
pPaste after cursor
ZZSave file (if modified) and exit Vim
ZQQuit without saving (like :q!)

Insert Mode

ShortcutAction
iInsert before cursor
IInsert at beginning of line
aInsert after cursor
AInsert at end of line
oOpen new line below
OOpen new line above
EscReturn to Normal mode

Visual Mode

ShortcutAction
vStart visual selection (character-wise)
VStart visual line selection
Ctrl+vStart visual block selection
yYank selection
dDelete selection
pPaste after selection

Command Mode (Colon Commands)

CommandAction
:wSave file
:qQuit
:wqSave and quit
:q!Quit without saving
:xSave and quit (same as :wq)
:e FILEOpen file
:r FILERead content of another file into current file
:%s/old/new/gReplace all occurrences of 'old' with 'new'
:/patternSearch for 'pattern'
:nohRemove search highlighting
:set numberShow line numbers
:set nonumberHide line numbers

Other Useful Shortcuts

ShortcutAction
Ctrl+oJump to older cursor position
Ctrl+iJump to newer cursor position
ggVGSelect all file content
zzCenter cursor line on screen
Ctrl+gShow file info (line, percentage)
. (dot)Repeat last command

Marks & Jumping

ShortcutAction
m[a-z]Set a mark named a–z at cursor position
'aJump to beginning of line of mark 'a'
`aJump to exact position of mark 'a'

Registers & Yanking

ShortcutAction
"ayYank (copy) into register a
"apPaste from register a
"0yYank into yank register 0 (default)

Macros

ShortcutAction
q[a-z]Start recording macro into register a–z
@aExecute macro stored in register a
@@Repeat last executed macro

Folding

ShortcutAction
zaToggle fold under cursor
zROpen all folds
zMClose all folds

Buffers & Windows

Shortcut / CommandAction
:bnGo to next buffer
:bpGo to previous buffer
:lsList all buffers
:sp fileOpen file in horizontal split
:vsp fileOpen file in vertical split
Ctrl+w h/j/k/lMove cursor between split windows

Search Enhancements

ShortcutAction
nRepeat last search in same direction
NRepeat last search in opposite direction
*Search for word under cursor forward
#Search for word under cursor backward

Visual Block Editing

ShortcutAction
Ctrl+vStart visual block mode
IInsert text at start of each line in block
AAppend text at end of each line in block
dDelete selected block
yYank selected block
