FCFS (First Come First Serve)


P1 → P2 → P3

First process arrives first.

FCFS Advantages

  • Simple
  • Easy To Implement

FCFS Disadvantages

  • Long Wait Time
  • Convoy Effect

SJF (Shortest Job First)


P1 = 10ms

P2 = 2ms

P3 = 5ms

Execution

P2 → P3 → P1

SJF Advantage

Minimum average waiting time.

SJF Disadvantage

Long jobs may starve.
