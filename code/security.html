🔐 PERL — ENCRYPTION (Complete Guide)

📌 1. What is Encryption?

Encryption is the process of converting plain text (readable data) into cipher text (unreadable data) using an algorithm and a key. 

Plain Text  →  Encryption Algorithm + Key  →  Cipher Text

Used for:

  • Passwords (secure storage)
  • API communication
  • Database protection
  • Confidential data (keys, tokens)

📌 2. Types of Encryption

🔹 2.1 Symmetric Encryption

Same key is used for encryption and decryption.

Encrypt → KEY → Decrypt
Examples:
  • AES (Advanced Encryption Standard)
  • DES
  • 3DES

🔹 2.2 Asymmetric Encryption

Uses two keys:

  • Public Key → Encrypt
  • Private Key → Decrypt
Examples:
  • RSA
  • ECC

$plain .= $gcm_dec->decrypt_done($tag);

print "Decrypted: $plain\n";

📌 7. GCM Mode — IV Concept

GCM (Galois Counter Mode) uses IV to generate a unique encryption stream. 

Ciphertext = Plaintext XOR Keystream
Keystream  = AES(Key + IV + Counter)

📌 8. Why IV MUST be UNIQUE (CRITICAL)

🚨 Problem Scenario

Same KEY + Same IV

C1 = P1 ⊕ K
C2 = P2 ⊕ K

C1 ⊕ C2 = P1 ⊕ P2  ❌
  • Attacker can recover plaintext
  • Authentication breaks
  • Encryption becomes useless

// routes/auth.js
const express = require("express");
const bcrypt = require("bcrypt");
const jwt = require("jsonwebtoken");
const db = require("../config/db");

const router = express.Router();

// REGISTER
router.post("/register", async (req, res) => {
  const { name, email, password } = req.body;

  const hash = await bcrypt.hash(password, 12);

  db.query(
    "INSERT INTO users (name, email, password) VALUES (?, ?, ?)",
    [name, email, hash]
  );

  res.json({ code: "000", message: "Success" });
});

// LOGIN
router.post("/login", (req, res) => {
  const { email, password } = req.body;

  db.query("SELECT * FROM users WHERE email=?", [email], async (err, result) => {

    if (result.length === 0)
      return res.json({ code: "001", message: "User not found" });

    const user = result[0];

    const valid = await bcrypt.compare(password, user.password);

    if (!valid)
      return res.json({ code: "002", message: "Invalid password" });

    const token = jwt.sign(
      { userId: user.id },
      process.env.JWT_SECRET,
      { expiresIn: "1h" }
    );

    res.json({ code: "000", token });
  });
});

module.exports = router;

📌 8. AUTH MIDDLEWARE


// middleware/authMiddleware.js
const jwt = require("jsonwebtoken");

module.exports = (req, res, next) => {

  const token = req.headers["authorization"];

  if (!token)
    return res.json({ code: "401", message: "No token" });

  try {
    const decoded = jwt.verify(token.split(" ")[1], process.env.JWT_SECRET);
    req.user = decoded;
    next();
  } catch {
    res.json({ code: "401", message: "Invalid token" });
  }
};
