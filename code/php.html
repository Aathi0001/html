PHP Basics

PHP (Hypertext Preprocessor) is a server-side scripting language used to build dynamic web applications. PHP code runs on the server and outputs HTML to the browser.

1. What is PHP?

PHP is a widely-used, open-source scripting language especially suited for web development. It can interact with databases, handle forms, manage sessions, and generate dynamic web pages.

  • Server-side scripting language
  • Embedded inside HTML
  • Executes on the server, not in the browser
  • Supports databases like MySQL, PostgreSQL, SQLite
  • Easy to learn and widely used

3. PHP Control Statements

3.1 if Statement

The if statement executes a block of code only when the condition is TRUE.

<?php
$age = 20;

if ($age >= 18) {
    echo "You are an adult";
}
?>

3.2 if-else Statement

Used when you want one block to run if TRUE, and another block if FALSE.

<?php
$age = 16;

if ($age >= 18) {
    echo "Adult";
} else {
    echo "Minor";
}
?>

3.3 else-if (elseif)

Used to check multiple conditions.

<?php
$marks = 75;

if ($marks >= 90) {
    echo "Grade A";
} elseif ($marks >= 70) {
    echo "Grade B";
} elseif ($marks >= 50) {
    echo "Grade C";
} else {
    echo "Fail";
}
?>

3.4 switch Statement

Used when comparing a variable with multiple fixed values.

<?php
$day = 3;

switch ($day) {
    case 1: echo "Monday"; break;
    case 2: echo "Tuesday"; break;
    case 3: echo "Wednesday"; break;
    default: echo "Invalid day";
}
?>

3.5 Loops

a) for loop

Used when you know the exact number of iterations.

<?php
for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";
}
?>

b) while loop

Runs as long as the condition is TRUE.

<?php
$i = 1;
while ($i <= 5) {
    echo $i . " ";
    $i++;
}
?>

c) do-while loop

Runs at least once even if the condition is FALSE.

<?php
$i = 1;
do {
    echo $i . " ";
    $i++;
} while ($i <= 5);
?>

d) foreach loop

Used to iterate over arrays.

<?php
$colors = ["Red", "Green", "Blue"];

foreach ($colors as $c) {
    echo $c . " ";
}
?>

4. PHP Functions

4.1 Creating a Function

Functions help reuse code by grouping logic into a single block.

<?php
function greet() {
    echo "Hello from PHP!";
}

greet();
?>

4.2 Function with Return Statement

A function can return a value using return.

<?php

    return $a + $b;

?>

4.3 Function Parameters

Values can be passed to functions through parameters.

<?php

    echo "Welcome, $name!";
}

welcome("Aathi");
?>

4.4 Default Parameters

Assign a default value to a parameter.

<?php
function greetUser($name = "Guest") {
    echo "Hello, $name!";
}

greetUser();          // Output: Hello, Guest!
greetUser("Aathi");   // Output: Hello, Aathi!
?>

4.5 Type Hints (Type Declarations)

Ensure parameter types (int, float, string, array, bool, object).

<?php

    return $a * $b;
}

echo multiply(4, 5);  // Output: 20
?>

4.6 Variable Functions

If a variable contains a function name, it can be called like a function.

<?php

    echo "Hello!";
}

$func = "hello";
$func();   // Calls hello()
?>

4.7 Anonymous Functions (Closures)

Functions without a name, often assigned to variables.

<?php
$sum = function($a, $b) {
    return $a + $b;
};

echo $sum(3, 7);  // Output: 10
?>

4.8 Arrow Functions (PHP 7.4+)

Short syntax for simple one-line anonymous functions.

<?php
$add = fn($a, $b) => $a + $b;

echo $add(10, 20);  // Output: 30
?>

4.9 Passing Functions as Arguments

Functions can be passed as callback values.

<?php

    echo $callback(5);
}

callMe(fn($x) => $x * 2);   // Output: 10
?>

5. PHP Arrays

Arrays store multiple values in one variable. PHP has three main types of arrays.

5.1 Indexed Arrays

Arrays with numeric indexes (0,1,2...).

Arrays inside arrays.

<?php

7. PHP Forms

Forms in PHP are used to collect user input from web pages. Data can be sent using GET or POST methods.

7.1 GET Method

Data is sent via URL. Visible in browser. Suitable for non-sensitive data.

<!DOCTYPE html>
<html>
<body>

<form method="get" action="process.php">
  Name: <input type="text" name="username"><br>
  <input type="submit" value="Submit">
</form>

</body>
</html>

<?php
// process.php
if(isset($_GET['username'])) {
    $name = $_GET['username'];
    echo "Hello, $name!";
}
?>

7.2 POST Method

Data is sent in HTTP request body. More secure. Suitable for passwords and sensitive data.

<form method="post" action="process.php">
  Password: <input type="password" name="password"><br>
  <input type="submit" value="Submit">
</form>

<?php
// process.php
if(isset($_POST['password'])) {
    $pass = $_POST['password'];
    echo "Password received.";
}
?>

7.3 Form Validation

Always validate user input before using it.

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    if (empty($_POST["username"])) {
        echo "Name is required";
    } else {
        $name = htmlspecialchars($_POST["username"]);
        echo "Hello, $name!";
    }
}
?>

7.4 $_REQUEST Superglobal

$_REQUEST contains data from GET, POST, and COOKIE. Can access any form data, but it's better to use $_GET or $_POST specifically.

<?php
// Using $_REQUEST
$name = $_REQUEST['username'] ?? 'Guest';
echo "Hello, $name";
?>

7.5 Summary

  • Use GET for non-sensitive data, URL parameters.
  • Use POST for sensitive data and larger data.
  • Always validate form inputs using PHP.
  • Use $_GET, $_POST over $_REQUEST for clarity and security.

9. PHP File Handling

PHP provides functions to work with files: reading, writing, appending, and uploading files via forms.

9.1 Reading Files

Using fopen(), fread(), fgets(), and file_get_contents().

<?php
// Using fopen() and fread()
$file = fopen("example.txt", "r"); // Open in read mode
$content = fread($file, filesize("example.txt"));
fclose($file);
echo $content;

// Using file_get_contents()
$content2 = file_get_contents("example.txt");
echo $content2;
?>

9.2 Writing Files

Using fopen() with write mode ("w") or file_put_contents().

<?php
// Using fopen() and fwrite()
$file = fopen("example.txt", "w"); // Write mode, overwrites
fwrite($file, "Hello PHP File Handling!");
fclose($file);

// Using file_put_contents()
file_put_contents("example.txt", "New content written!");
?>

9.3 Appending to Files

Use "a" mode in fopen() or file_put_contents() with FILE_APPEND.

<?php
// Using fopen() in append mode
$file = fopen("example.txt", "a");
fwrite($file, "\nAppended line.");
fclose($file);

// Using file_put_contents() with FILE_APPEND
file_put_contents("example.txt", "\nAnother appended line.", FILE_APPEND);
?>

9.4 File Uploads

Using $_FILES superglobal and HTML form.

<form method="post" enctype="multipart/form-data">

9.5 File Functions Summary

  • fopen() ‚Äì Open a file in read/write/append mode
  • fread() ‚Äì Read file content
  • fwrite() ‚Äì Write to a file
  • fclose() ‚Äì Close a file
  • file_get_contents() ‚Äì Read entire file content at once
  • file_put_contents() ‚Äì Write or append to a file
  • move_uploaded_file() ‚Äì Move uploaded file to a directory

10. PHP ‚Äì Object-Oriented Programming (OOP)

OOP in PHP allows you to create classes, objects, and use concepts like inheritance, interfaces, and traits for better code structure and reusability.

10.1 Classes & Objects

<?php
// Class definition
class Person {
    public $name;
    public $age;

    // Method
    public function introduce() {
        echo "Hello, my name is $this->name and I am $this->age years old.";
    }
}

// Creating object
$person1 = new Person();
$person1->name = "Aathi";
$person1->age = 21;
$person1->introduce();
?>

10.2 Constructor

A constructor is a special method called automatically when an object is created.

<?php
class Person {
    public $name;
    public $age;

    // Constructor
    public function __construct($name, $age) {
        $this->name = $name;
        $this->age = $age;
    }

    public function introduce() {
        echo "Hello, I am $this->name, $this->age years old.";
    }
}

$person2 = new Person("Rahul", 25);
$person2->introduce();
?>

10.3 Inheritance

A class can inherit properties and methods from another class using extends.

<?php
class Animal {
    public function eat() {
        echo "Eating...";
    }
}

class Dog extends Animal {
    public function bark() {
        echo "Barking...";
    }
}

$dog = new Dog();
$dog->eat();   // Inherited from Animal
$dog->bark();  // Own method
?>

10.4 Interfaces

Interfaces define methods a class must implement. They enforce a contract.

<?php
interface Logger {
    public function log($message);
}

class FileLogger implements Logger {
    public function log($message) {
        echo "Logging to file: $message";
    }
}

$logger = new FileLogger();
$logger->log("Hello World");
?>

10.5 Traits

Traits are used to reuse methods across multiple classes.

<?php
trait Hello {
    public function sayHello() {
        echo "Hello from Trait!";
    }
}

class User {
    use Hello;
}

$user = new User();
$user->sayHello();
?>

10.6 Static Members

Static properties and methods belong to the class, not objects. Access using :: operator.

<?php
class Math {
    public static $pi = 3.14;

    public static function square($x) {
        return $x * $x;
    }
}

echo Math::$pi;           // Access static property
echo Math::square(5);     // Access static method
?>

10.7 Summary

  • Classes define blueprint, objects are instances.
  • Constructor (__construct) initializes objects.
  • Inheritance (extends) allows code reuse.
  • Interfaces define method contracts.
  • Traits allow method reuse across classes.
  • Static members belong to the class itself, not objects.

11. PHP ‚Äì Error Handling

PHP provides ways to handle runtime errors and create custom error messages for better control.

11.1 try / catch

Used to catch exceptions and handle them without breaking the application.

<?php
function divide($a, $b) {
    if ($b == 0) {
        throw new Exception("Division by zero.");
    }
    return $a / $b;
}

try {
    echo divide(10, 2);  // Output: 5
    echo divide(5, 0);   // Throws exception
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
?>

11.2 finally

Optional block to execute code regardless of exception.

<?php
try {
    echo divide(10, 0);
} catch (Exception $e) {
    echo "Caught: " . $e->getMessage();
} finally {
    echo " - Execution finished.";
}
?>

11.3 Custom Error Handling

Create your own error handler using set_error_handler().

<?php
// Custom error function
function myErrorHandler($errno, $errstr, $errfile, $errline) {
    echo "Custom Error [$errno]: $errstr in $errfile on line $errline";
}

// Set custom error handler
set_error_handler("myErrorHandler");

// Trigger an error
echo $undefinedVar;  // Notice triggers custom handler
?>

11.4 trigger_error()

Manually trigger user-level errors.

<?php
$value = -1;

if($value < 0) {
    trigger_error("Value must be positive", E_USER_WARNING);
}
?>

11.5 Summary

  • Use try/catch to catch exceptions safely.
  • throw keyword is used to generate exceptions.
  • finally block always executes.
  • Use set_error_handler() for custom error handling.
  • trigger_error() allows manual error triggering.

12. PHP ‚Äì PDO (PHP Data Objects)

PDO is a database access layer providing a uniform method to interact with databases like MySQL, PostgreSQL, etc. It supports prepared statements for secure queries.

12.1 Connecting to MySQL

<?php
$host = 'localhost';
$db   = 'testdb';
$user = 'root';
$pass = '';
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";

try {
    $pdo = new PDO($dsn, $user, $pass);
    // Set PDO error mode to exception
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connected successfully";
} catch (PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}

13. PHP Security

Security is essential to protect applications from attacks like SQL Injection, XSS, and insecure password storage.

13.1 SQL Injection

Occurs when user input is directly included in SQL queries.

<?php
// Unsafe (vulnerable)
$user = $_GET['username'];
$sql = "SELECT * FROM users WHERE username='$user'";
$pdo->query($sql);

// Safe using prepared statements
$stmt = $pdo->prepare("SELECT * FROM users WHERE username = :username");
$stmt->bindParam(':username', $user);
$stmt->execute();
?>

13.2 Cross-Site Scripting (XSS)

Occurs when user input is rendered directly in HTML.

<?php
// Unsafe
echo $_GET['comment'];

// Safe
echo htmlspecialchars($_GET['comment']); // Converts special characters to HTML entities
?>

13.3 Password Hashing

<?php
$password = "mypassword";

// Hash password
$hash = password_hash($password, PASSWORD_DEFAULT);

// Verify password
if(password_verify("mypassword", $hash)) {
    echo "Password matched";
} else {
    echo "Invalid password";
}
?>

13.4 Summary

  • Always use prepared statements to prevent SQL Injection.
  • Use htmlspecialchars() to prevent XSS attacks.
  • Never store plain text passwords; always hash using password_hash().

Data Encryption in PHP (AES & RSA ‚Äì Deep Study)

Encryption is the process of converting readable data (plaintext) into unreadable data (ciphertext) to protect it from unauthorized access.

PHP mainly supports two types of encryption:

  • Symmetric Encryption ‚Üí Same key for encrypt & decrypt (AES)
  • Asymmetric Encryption ‚Üí Public & private keys (RSA)

1. AES Encryption (Symmetric Encryption)

AES (Advanced Encryption Standard) uses the same secret key for both encryption and decryption. It is fast and commonly used for: passwords, tokens, session data, files.

Encryption Flow

  • Plain text
  • Secret key
  • Initialization Vector (IV)
  • Cipher method (AES-256-CBC)

AES Encryption Example


$data = "My Secret Message";
$key = "my_secret_key_123456";
$cipher = "AES-256-CBC";

$ivLength = openssl_cipher_iv_length($cipher);
$iv = openssl_random_pseudo_bytes($ivLength);

$encrypted = openssl_encrypt(
  $data,
  $cipher,
  $key,
  0,
  $iv
);

$encryptedData = base64_encode($iv . $encrypted);
echo $encryptedData;

The output is encrypted binary data encoded in Base64 so it can be safely stored or transmitted.
